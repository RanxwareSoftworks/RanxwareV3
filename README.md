# RanxwareV3
The official Ranxware V3 repository

### Loadstring
Use this line of code to execute the script:
```luau
loadstring(game:HttpGet("https://github.com/RanxwareSoftworks/RanxwareV3/raw/refs/heads/main/main.luau"))()
```
